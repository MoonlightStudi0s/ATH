# AutoTouristicHelper — Умный помощник в автопутешествиях

**AutoTouristicHelper** — это веб-приложение для автомобилистов, которое с помощью искусственного интеллекта строит оптимальные, интересные и безопасные маршруты по всей России. Проект интегрирует логистические точки сети АЗС «Роснефть», лучшие региональные отели и уникальные достопримечательности, превращая обычную поездку в увлекательное геймифицированное приключение.

Приложение оформлено в строгом, чистом корпоративном дизайне с интуитивно понятным интерфейсом и интерактивной картой.

##  Основные возможности

1. **ИИ-Генератор маршрутов (Groq API):**
    
    - Распределяет поездку на выбранное количество дней (от 2 до 7).
        
    - Исключает зигзаги и нелогичные крюки на трассе.
        
    - На каждый день ИИ предлагает: минимум одну проверенную **АЗС «Роснефть»** (с расчётом расхода топлива, стоимости заправки и доступного сервиса), **достопримечательности** для культурной программы и **отель** для безопасного ночлега.
        
2. **Интерактивная карта (на Leaflet.js):**
    
    - Автоматический геокодинг (Nominatim OSM) транзитных точек.
        
    - Рендеринг сглаженной линии трека (`LineString`) и интерактивных маркеров с кастомной цветовой кодировкой типов мест.
        
    - Микро-рандомизация координат (предотвращает наложение меток друг на друга в пределах одного населенного пункта).
        
    - Подробные поп-апы с полной информацией по каждому объекту при клике.
        
3. **Геймификация и программа лояльности:**
    
    - **Баланс коинов (🪙):** Накапливается за создание маршрутов и выполнение дорожных чекпоинтов.
        
    - **Чекпоинты:** Интерактивные квесты («Первый заезд», «Марафонец» на 100 км пути) с возможностью забрать заработанную награду прямо в интерфейсе.
        
    - **Магазин мерча:** Фирменная одежда и сувениры от Роснефть x ATH. Реализована система проверки баланса перед покупкой и вкладка **«Купленное»** для просмотра приобретенных товаров.
        
4. **Личный кабинет и безопасность:**
    
    - Полноценная регистрация и авторизация на сессиях (`flask.session`).
        
    - Безопасное хэширование паролей в БД при помощи алгоритмов `werkzeug.security`.
        
    - Динамический профиль с возможностью редактирования имени в реальном времени.
        
    - Реальная история поездок — отображаются только те маршруты, которые пользователь действительно сгенерировал под своей учетной записью.
        

##  Стек технологий

- **Backend:** Python 3.11+, Flask, SQLite 3, Werkzeug (Security), OpenAI SDK.
    
- **LLM Engine:** Groq API (модель `openai/gpt-oss-120b`).
    
- **Frontend:** HTML5, CSS3 (модернизированный строгий стиль без лишнего визуального шума), Vanilla JS (без тяжелых фреймворков), Leaflet.js (интерактивные карты).

##  Установка и локальный запуск

### 1. Подготовка окружения

Убедитесь, что у вас установлен Python версии 3.10 или выше.

Распакуйте архив.

Склонируйте проект и установите зависимости:

```
pip install flask openai werkzeug
```

### 2. Настройка API-ключа ИИ

В файле `app.py` убедитесь, что указан действующий ключ авторизации API Groq:

```
GROQ_API_KEY = "ваш API key"
```

### 3. Запуск сервера

Запустите бэкенд из консоли:

```
python app.py
```

После запуска база данных `roadhelper.db` и все необходимые таблицы инициализируются автоматически.

## Хостинг внутри локальной сети (Wi-Fi)

По умолчанию Flask запускает приложение в приватном режиме на адресе `127.0.0.1:5000`. В `app.py` настроен запуск на внешнем хосте:

```
if __name__ == '__main__':
    app.run(debug=True, host='0.0.0.0', port=5000)
```

Чтобы подключить к тестированию другие устройства (телефоны, планшеты, ноутбуки):

1. Подключите все устройства к одной Wi-Fi сети.
    
2. Узнайте локальный IP-адрес хост-компьютера. В Windows выполните команду:
    
    ```
    ipconfig
    ```
    
    Найдите строку `IPv4-адрес` (например, `192.168.1.15`).
    
3. Откройте браузер на мобильном телефоне и перейдите по адресу:
    
    ```
    [http://192.168.1.15:5000](http://192.168.1.15:5000)
    ```
    
    _(Замените `192.168.1.15` на ваш фактический локальный IP)._
